## 프로젝트 실행 방법

#### 모듈 설치
`npm install` 또는 `npm i`

#### 실행
`npm run dev`

## 사용한 라이브러리
- 스타일: **Tailwind CSS**
- 아이콘: **React Icons**
- API 통신: **Axios**
- 유효성 검증: **Zod**
- 스크롤 이벤트 감지: **React Intersection Observer**
- 데이터 패칭: **Tanstack Query**
- 토스트 메시지: **React Toastify**
- 코드 스타일링: **Prettier**
- 페이지 라우팅: **React Router Dom**
